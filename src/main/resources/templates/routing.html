Controller 연결방법 

    1. 화면 Redering은 return 값을 반드시 String 타입으로 던져야 Template Engine파일과 연결 됩니다.
        예) a href=/link => link.html 파일을 templates 폴더에 생성해야 합니다.
    2. Json return 타입은 RestController를 이용 합니다.
    

    @GetMapping("/{path}")
    public String onePath(@PathVariable String path, Model model) {
        model.addAttribute("title", path + " title");
        return path;
    }
