Exception Handling
Exception 관리

Exception Handling 관리에 대한 문서 입니다.

강제로 Rollback이 필요할 경우는 아래와 같이 BusinessException을 발생하면 됩니다.


        // 사용예시
        throw new BusinessException("ERR01", "ROLLBACK");

        // 구현체
        // stacktrace을 받는 경우
        public BusinessException(Throwable throwable) {
            super(throwable);
        }

        // 코드, stacktrace을 받는 경우
        public BusinessException(String code, Throwable throwable) {
            super(code, throwable);
        }

        // 코드, 정의된 메시지를 받는 경우
        public BusinessException(String code, String msg) {
            super(code, msg);
        }
